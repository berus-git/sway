# Sway ablakkezelő
- Sway konfigurációs állomány: config (~/.config/sway/config).
- Sway konfigurációs állomány (KDE alapokon): config_kde (~/.config/sway/config).
- Swaybar indító állomány: status.sh (~/.config/sway/status.sh).
- Szükséges változók, hogy a KDE alkalmazások megfelelően jelenjenek meg: environment (pl. ~/.profile).
